Create a Vue node project to store shared styles and components.

### Styles
Create a global, default theme. Each site will have the ability to make their own overrides for things like colors.

### Components
We'll include the components for the following common use cases:
- Card
- Dropdown
- InputText
- FormGroup
- ZachZoneSwitcher (See [[Add side nav for other sites]])
- Navbar
- Badge
- Toast & Toast Container
