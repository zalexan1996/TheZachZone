
> [!summary] Story
> I should be able write reviews for games in markdown.

> [!important] Out of Scope
> - Editing a review
> - Deleting a review
> - Viewing the review on the game page

> [!done] Acceptance Criteria
> - I should be able to write a review for a game if I'm logged in.
> - I can score the game up to 5 stars
> - My review stores the total time spent playing at the time of the review, as well as total time playing of all time.


> [!info] Implementation Details
> - Create a migration to create the review table and link it to the Game and User tables
> - Find a Nuget package for working with markdown
> - Create or find an existing markdown editor UI component
